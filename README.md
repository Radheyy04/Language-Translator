# Language-Translator
This is a web-based Language Translator application built using HTML, CSS, and JavaScript that translates text between multiple languages using a translation API. It features real-time translation, a clean and modern UI, language selection, and a responsive interface.

## Features
- Real-time text translation
- Multiple language support
- Language exchange option
- Responsive interface
- Interactive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Translation API
