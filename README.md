# Language Translator Web App

A responsive and interactive language translator web application designed and developed using HTML, CSS, and JavaScript.
The project features a custom modern UI along with real-time translation, speech recognition, text-to-speech, language swapping, auto language detection, and clipboard copy functionality using browser APIs and Google Translate API.

## Features
- Real-time language translation
- Speech recognition (Voice Input)
- Text-to-speech support
- Language exchange/swap
- Copy translated text
- Auto detect source language
- Fully responsive custom UI

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Translation API
